IMPORTANTE !

Dados dos Depoimentos vazios por pedido do cliente, para acrescentar/alterar os depoimentos vá para a pasta "Textos" e altere os aquivos dentro das pastas "Cliente01", "Cliente02" ou "Cliente03" => "depoimento.txt" para o depoimento do cliente, e "nome.txt" para o autor do comentário.

Dentro da pasta "Textos", também é possível alterar o texto da seção "Quem Somos", na pasta "quem_somos" no arquivo "quem_somos.txt".

A configuração de envio de email pelo formulário contate-nos está em "Templates/sendmail.php".

O email do cliente, ou seja quem recebe as solicitações está definido na variável MAIL_CLIENTE que se encontra em "/fun/init.php".
